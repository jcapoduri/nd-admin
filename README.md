# nd-admin
neodymium project instance administrator

### How to work with this ###
* Compile
after clone : `npm install`
after changes on UI : `gulp compile`
clean up db:
  * linux   : `php tools/setup.php`
  * windows : `[path_to_php.exe] tools/setup.php`
